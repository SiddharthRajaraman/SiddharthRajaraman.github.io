    Intro to Data Types

    ×

    In programming, data is simply the pieces of information that we use to create programs

    • E.g. the numbers in a calculation
    • E.g. the text printed on the screen

    Data types are essentially different types of data. the type of data provides us with information such as how it is storied and what operations can be performed on it

    • E.g. we can multiply numbers but we cannot multiply words

    Primitives: the most basic data types (Integers, Floats, Strings, Booleans)


    Floats and Integers

    In programming, Integers and Floats are used to do three main things

    • Descriptions - What is someone's age? What is the average test grade
    • Calculations - what is the sum of two numbers? What is the distance from the earth to the moon
    • Counting - how many times did the car drive around the track? How many times did the rocket fly around the world?

    Integers vs. Floats

    We all know that in math, there are integers and there are decimals. While one is a whole number, the other represents a whole number PLUS a fraction of a number

    In programming, the concept remains the same. While int represents an integer, float represents a decimal


    Strings

    Strings: a sequence of characters

    Characters: letters, spaces, numbers, or symbols

    Strings are used to:

    • Display data to the screen
    • Add or remove text (we can split strings into smaller strings or even combine strings to make longer ones)
    • Modify characters (E.g. we could capitalize the first letter in a string)
    • Allow the computer to communicate with us in a human friendly manner.

    Strings can look like other data types. It's important to knowt hat even if a number looks like a number, it might be a string

    For example, there is a difference between '35' and 35


    Boolean

    Boolean values represnt true and false

    Booleans are used to:

    • Validate things (did a youtube video go viral?)
    • make decisions (is it cold outside? Yes (true)? Then I should wear a sweater)

    The two boolean values are opposites. Sometimes, we can think of them as on or off, or even 1 or 0

    It is important to know that true and false are not strings


    Lists and Tuples

    Lists are simply collections of different objects with no defined size

    They are mutable, which means that they can be modified

    They have many different uses. For example, they can be used to:

    • List files in a folder
    • List employees in a company
    • List emails in your inbox

    Tuples, like lists are collections of different objects. The difference, is that while lists have no set size, tuples do.

    While you can edit lists, you cannot edit tuples.

    The only syntax difference between lists and tuples is while lists have brackets, tuples have parentheses


    2D Lists - Nesting

    Python supports nesting. One common example of this is to have a list that contains other lists

    As we can see, these 2D Lists can serve as game boards as well as other things


    Dictionaries

    Python dictionaries are known as mappings

    • Mappings are also collections of objects (strings, ints, floats etc.)
    • They store objects by key rather than by position
    • Dictionaries have no defined order (such as first to last)

    Dictionaries are also mutable

    Converting between Data Types

    ×

    Type Conversion: converting from one data type to another

    There are two types of type conversion

    • Implicit Type Conversion: Python automatically converts from one data type to another
      • E.g. when adding an integer and a float, the sum is a float
    • Explicit Type Conversion: the user converts from one data type to another
      • If you ahve the integer: 35 and the string: "45", you can add them by converting the string to an integer and finding the sum of both numbers

    For Loops

    ×

    In python, for loops are usually used to iterate through lists, strings and tuples


    Range()

    The range function essentially creates a sequence. For example, range(100) will create a sequence from 0 to 99 (0, 1, 2, 3, ...99)

    The range function can also create a sequence with a specific start, stop, and step size

    You can also use the range function with the len function within for loops


    Using 'else' in for loops

    A for loop can also include an else statement, which is only executed when the for loop stops executing


    The 'break' command

    The break command immediately leaves the function

    While Loops

    ×

    A while loop is used to repeatedly execute a block of code until a condition is met

    As opposed to a for loop, a while loop is commonly used when there is no predefined number of iterations necessary.


    Using 'else' in while loops

    The else portion will only execute if the while loop's condition becomes false

    If a break statement is used in the while loop, the else statement will not execute no matter what
